## 接入美的BIoT平台


## 环境要求
- 您的系统需要达到 [环境要求][Requirements], 例如，安装了不低于 1.10.x 版本的 Go 环境。

## 安装
使用 `go get` 下载安装 SDK

```sh
$ go get -u github.com/shanweidi/midea-BIoT-sdk-go/sdk
```
